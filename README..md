# Library Management System - Backend

## Tech Stack

- Node.js + Express + TypeScript
- MongoDB + Mongoose
- JWT + RBAC
- Clean Architecture

## Available Endpoints

- **Auth**: `/api/auth/register`, `/api/auth/login`
- **Books**: `/api/books`
- **Members**: `/api/members`
- **Borrowing**: `/api/borrowings/issue`, `/api/borrowings/return/:id`
- **Dashboard**: `/api/dashboard/stats`
- **API Docs**: `/api-docs`

## Development

```bash
npm run dev
```
